# Operations Guide

## Local Run

```bash
streamlit run app.py
```

## Refresh GenAI Region Availability

```bash
python scripts/setup.py
```

Run this whenever OCI region availability, subscriptions, or GenAI model availability may have changed.

## Metadata

Local metadata is stored in:

```text
data/metadata/
```

Generated Markdown reports are stored in:

```text
data/reports/
```

Uploaded working files are stored in:

```text
data/uploads/
```
